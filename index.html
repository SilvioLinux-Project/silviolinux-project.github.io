Sobre o SilvioLinux

SilvioLinux é uma coleção de edições customizadas — LFS (Linux From Scratch), Arch, Gentoo, Debian e BusyBox — pensada para máquinas com recursos limitados ou para quem gosta de configurar tudo manualmente.

Comandos de atualização

  • LFS (scratchpkg): sudo scratchpkg sync Base / sudo scratchpkg upgrade Base
  • Gentoo: sudo emerge-websync Base
  • Debian: sudo apt-get update / sudo apt-get upgrade
  • Arch: sudo pacman -Sy / sudo pacman -Syu
  • BusyBox (silviobuilds): sudo silviobuilds --sync Base / sudo silviobuilds --upgrade Base
