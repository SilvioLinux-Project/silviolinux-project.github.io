LFS (Linux From Scratch)

  • Versão com instalador gráfico Calamares — 32-bit (x86/i686) — Requisitos: mínimo 512MB RAM
  • Versão com instalador gráfico Calamares — 64-bit (x86_64/amd64) — Requisitos: mínimo 768MB RAM
  • Versões modo texto (ncurses) 32-bit: 96MB RAM / 64-bit: 256MB RAM
  • Comandos: sudo scratchpkg sync Base / sudo scratchpkg upgrade Base

Arch Linux (base)

  • Instalador gráfico Calamares — apenas 64-bit (x86_64/amd64) e ARM64 (AArch64)
  • Comandos: sudo pacman -Sy / sudo pacman -Syu

Gentoo

  • Instalador gráfico Calamares — suportes: 32-bit x86/i686, 64-bit x86_64/amd64, PowerPC 32-bit (PPC), PowerPC 64-bit (PPC64)
  • Comando: sudo emerge-websync Base

Debian (13)

  • Instalador gráfico Calamares — suportes: 64-bit x86_64/amd64 e PowerPC (PPC64)
  • Comandos: sudo apt-get update / sudo apt-get upgrade

BusyBox

  • Instalador ncurses 32-bit: roda com 8MB RAM
  • Instalador ncurses 64-bit: roda com 128MB RAM
  • Suporte: x86/i686, x86_64/amd64, PowerPC (PPC, PPC64)
  • Comandos: sudo silviobuilds --sync Base / sudo silviobuilds --upgrade Base
